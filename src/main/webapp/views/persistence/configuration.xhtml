Create a class that extends com.xpert.persistence.dao.BaseDAOImpl. The implementation of DAO classes must extends this class.
This class must set the Entity Manager. Calling super.setEntityManager.

To create interfaces for DAOs, just create a class that extend com.xpert.persistence.dao.BaseDAO. 
                    import javax.persistence.EntityManager;
                    import javax.persistence.PersistenceContext;

                    /**
                    *
                    * @author Ayslan
                    */
                    public class BaseDAOImpl<T> extends com.xpert.persistence.dao.BaseDAOImpl<T> {

                        @PersistenceContext(unitName = "xpertShowcasePU")
                        private EntityManager entityManager;

                        public BaseDAOImpl() {
                        }

                        @Override
                        public EntityManager getEntityManager() {
                            return entityManager;
                        }
                    }
