Restrition Collector é um componente para adicionar restrições em Listas e em LazyDataModel.
Existem 2 componentes para isso o x:restriction e x:restrictionCollector

x:restriction

Usado para definir a restrição, nele pode ser configurado o nome da propriedade e outros parâmetros. 
                <p:inputText>
                    <x:restriction property="name" type="like" />
                </p:inputText>

x:restrictionCollector

Este componente é um ActionListener, ele define quem receberá as restrições. 
                <p:commandButton value="Filter" >
                    <x:restrictionCollector addTo="\#{restrictionComponentMB.dataModelPerson}" />
                </p:commandButton>
